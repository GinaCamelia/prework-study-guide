Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • We use the add -A command to commit all files in the working branch.
  • git commit -m "add comment here" commits the changes;
    We use the -m flag to associate a message with our commit. Commit messages should be short descriptive messages of the changes implemented. Once completed, our Git repository is now up to date with our working directory!
  • git pull origin main - It is a good habit to always pull the base branch into your feature branch before pushing up any code.
  • git push origin feature/add-html - pushing all the changes
